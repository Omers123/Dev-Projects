Coding Cheat Sheet


Tag Explanation
<table> Creates the table
<thead> Header section of the table
<tr> Table Row
<th> Table header
<td> Table data cell
<tbody> Main body of the table
<!DOCTYPE> Defines the docuiment type
<a> Defines a hyperlink
<body> Defines the document's body
<button> Defines a clickable button
<div> Defines a section in a document
<em> Defines a emphasized text
<embed> Defines a container for an external aplication
<footer> Defines a footer for a document or section
<head> Contains metadata/information for the document
<header> Defines a header for a document or section

Make sure to add more tags in here for help in the future!
